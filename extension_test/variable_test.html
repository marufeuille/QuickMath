QuickMath 変数計算テスト

このページでは、QuickMath拡張機能の変数計算機能をテストできます。

使い方:

  1. 以下の式や方程式を選択します
  2. 右クリックして「数式を計算する」を選択します
  3. 変数を含む式の場合は、変数の値を入力すると結果が表示されます
  4. 方程式の場合は、自動的に解が表示されます

1. 単一変数の式

2 * x + 5

期待される動作: 変数xの入力フィールドが表示され、値を入力すると結果が計算されます。

2. 複数変数の式

x * y + z

期待される動作: 変数x、y、zの入力フィールドが表示され、すべての値を入力すると結果が計算されます。

3. 単純な方程式

2 * x + 5 = 15

期待される動作: 方程式が解かれ、x = 5 と表示されます。

4. 二次方程式

x^2 - 4 = 0

期待される動作: 方程式が解かれ、x = 2 と x = -2 の2つの解が表示されます。

5. 別の二次方程式

x^2 + 2*x + 1 = 0

期待される動作: 方程式が解かれ、x = -1 の重解が表示されます。

6. 通常の数式（変数なし）

12 + 34 * 2

期待される動作: 従来通り計算結果（80）が表示されます。

サポートされている方程式のタイプ

  • 線形方程式: ax + b = c の形式（例: 2x + 5 = 15）
  • 二次方程式: ax² + bx + c = 0 の形式（例: x² - 4 = 0）

注意: より複雑な方程式は現在サポートされていません。
